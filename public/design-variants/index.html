Aperture Flow — Design Variants

10 layout explorations for the v1 native app. Same information architecture, different visual approaches. 3 avant-garde, 4 iterative, 3 completely random.

Shared Information Architecture

  • Now: M-F day columns with meetings + tasks
  • Today highlighted, past days dimmed
  • Undone past tasks show amber "still open" badge
  • Task completion: checkbox, strikethrough, 2s undo
  • Next (2 weeks): cards without day assignment
  • Later: compact backlog, collapsed by default
  • Flash cards: 320px, person/initiative + topic list
  • Triage: K-Now, A-Name, D-Name, T-Name, X

Avant-Garde

Bold, unconventional layouts that push boundaries

01 Radial Timeline

Days orbit a central "today" hub in concentric rings. Radial distance indicates time horizon — center is most urgent. Smooth orbital animation on load.

burnt orange / gold on dark Playfair Display + Source Code Pro orbital layout
View

02 Overlapping Cards

Stacked, overlapping card deck with asymmetric layout and depth. Day columns peek from behind each other with 3D transforms on hover and floating stagger animation.

cool silver / blue on light GT America + IBM Plex Mono 3D depth + parallax
View

03 Brutalist Axis

Geometric precision, pure monochromatic grid. Two-axis matrix (horizontal = priority, vertical = days). No curves, border-driven structure, maximum information density.

black / white / yellow JetBrains Mono + Courier Prime data visualization
View

Iterative

Polished refinements of the M-F column pattern

04 Warm Neutrals

The "notebook" feel. Sand, cream, and charcoal with warm amber accents. Serif headers give an editorial quality. Equal 5-column grid with spacious cards.

05 Cool Professional

Clean dashboard polish. Slate grays with blue accents on white. Modern sans-serif with monospaced time displays. Professional hierarchy, trustworthy feel.

06 Dark Mode

Dark background with muted cyan, lime, and orange accents. High contrast text on dark, comfortable for extended use. Technical, modern feel.

Bold red accent on neutral with a dark sidebar housing flash cards and backlog. 3-column day grid + right sidebar. Editorial, print-magazine energy.

Random

Completely unexpected, wildcard approaches

Inspired by music production software (Ableton/DAW). Days as horizontal tracks with meetings and tasks as draggable clips positioned in time. Playhead shows current time with pulsing glow.

09 Concentric Rings

Days plotted as concentric rings radiating from center (today = innermost). Tasks rendered as glowing dots by day + time. People as orbital cards floating around the canvas. Abstract, zen.
